# Student Performance Dashboard - Tableau

This repository contains an interactive Tableau dashboard designed to analyze student performance based on multiple lifestyle factors like study hours, sleep, diet quality, mental health, social media usage, and more.

## üìä Dashboard Features

### ‚úÖ Key Highlights
- **Average Metrics** displayed at the top (e.g., Exam Score, Social Media Hours, Netflix Hours, Mental Health Rating, Attendance Percentage).
- **Dynamic Area Chart** showing the correlation between a user-selected measure (e.g., Study Hours or Sleep Hours) and Exam Score.
- **Interactive Heatmaps** visualizing the combined impact of:
  - **Study Hours & Diet Quality** on various metrics.
  - **Sleep & Diet Quality** on Attendance Rate or Exam Score.
- **Gender-wise Distribution** of Exam Scores using proportional circles.
- **Bar Chart** analyzing the impact of chosen dimensions (e.g., Diet, Parental Support, Extra Curricular Activities) on Exam Score.
- **Histogram** showing the distribution of students based on binned Exam Scores.

---

## üß© Interactivity & Parameters

This dashboard is highly interactive and allows users to personalize the view through several parameter controls:

- **Measure Selector (Area Chart)**: Choose between `Study Hours` or `Sleep Hours`.
- **Dimension Selector (Top Heatmap)**: Switch between `Diet Quality` and `Mental Health`.
- **Measure Selector (Bottom Heatmap)**: Choose between `Attendance Rate` or `Exam Score`.
- **Dimension Selector (Bar Chart)**: Analyze Exam Score based on:
  - `Diet Quality`
  - `Parental Support`
  - `Extra Curricular Activities`

Dynamic titles in the charts update automatically based on selected parameters, making the insights more relevant and contextual.

## üí° Purpose

The purpose of this dashboard is to provide a visual and analytical understanding of how different lifestyle habits influence student performance. This can be helpful for educators, students, and policymakers to make data-driven decisions.
